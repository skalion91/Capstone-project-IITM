# Project Name
- Project 1 – Interconnected Vehicles and Proactive Risk Management

## Overview
 - While connected vehicles are presently engulfed by the core problem of relying on a reactive risk management of vehicle related incidents. This domain presents a challenge of IoT data from several sensors from huge fleet of connected vehicles thereby producing a massive dataset.

 1.	Detect and prevent issues early that will reduce the number of vehicles related incidents significantly;
 2.	Continuously monitor risky driver behaviours, driving conditions, ADAS  warnings, and vehicle conditions related data; create a model that will help create an ecosystem of safer transportation systems
 3.	Detect anomalies, classify and label risky drivers proactively
 

## Workflow
- EDA & Data Insights
- Preprocessing & Imbalance Handling
- Classification Model Training & Evaluation
- Deployment Pipeline

## Expected Deliverables
- EDA Report
- Cleaned Dataset
- Feature engg Catalogue
- Trained Model
- Evaluation Report
